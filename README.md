# Fantasy
#### Fantasy是基于.NET的高性能网络开发框架，支持主流协议，前后端分离。
#### 支持Unity/Godot/WinFrom/WPF/Console等C#客户端接入。
#### 框架支持TCP\KCP\WebSocket\http（支持Unity发布成H5）三种网络协议。 
#### 适合需要快速上手、可扩展、分布式全平台商业级解决方案的框架。
